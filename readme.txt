== Description ==

[WP Picture Calendar](http://www.ism-soft.com/tools/wp-picture-calendar-plugin) is a wordpress plugin that allows you to add new widget to Your sidebar. 
WP Picture Calendar shows a simple image calendar of World Days whith date and short description. With WP Picture Calendar you will not miss World Day anymore.

Description:

1. All of the photograph link (nofallow) to the relevant Wordl Day pages , so anyone interested can learn more about the World Day shown. 
2. Photo is described by a two-line description under the photo.
3. Plugin adds the contents with the best practices of SEO. 
4. Widget shows current World Day or countdown to the next random World Day.

Demo :

You can see a demo on my website : [Jakubas.net.pl](http://jakubas.net.pl/test/)

== Installation ==

    * Download it 
    * Extract all files from the zip archive
    * Copy the wp-picture-calendar folder to wp-content/plugins
    * Activate the plugin through the 'Plugins' menu in wordpress
    * Add WP Picture Calendar Widget to your sidebar
    * That's all !
